How to See the Full Blog Page

This test page embeds the blog data directly. To see the full blog system with dynamic loading:

Option 1: Quick Local Server (Easiest)

Open terminal in your project folder and run:

python3 -m http.server 8000

Then visit: http://localhost:8000/blog.html

Option 2: Use VS Code Live Server

If you use VS Code, install the "Live Server" extension, right-click blog.html, and select "Open with Live Server".

Option 3: Deploy to GitHub Pages

Push to GitHub and enable Pages in repository settings. It will work perfectly there!
